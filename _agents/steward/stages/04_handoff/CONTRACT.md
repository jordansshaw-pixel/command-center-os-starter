# Steward Handoff Stage

Status: Initial stage contract
Date: 2026-06-05

## Purpose

Move governance state to the next owner.

## Handoff Routes

- Execution or routing -> Conductor.
- Memory/correction -> Keeper, Recorder, Librarian.
- Voice after truth approval -> Voice.
- Stop/escalation -> the operator.

## Required Signal

Use `_agents/signal/SIGNAL-STANDARD.md` when work moves.
