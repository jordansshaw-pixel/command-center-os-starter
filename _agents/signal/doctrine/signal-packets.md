# Signal Doctrine: Signal Packets

Status: jurisdictional doctrine
Date: 2026-06-07

## Load When

Load when work moves between roles, stages, projects, sessions, reviews, builds, stops, or corrections.

## Authority

Signal packages state so the next owner can act without guessing.

## Must

- Name active agent, active stage, source, audience, workspace, risk, status, reason, next action, memory impact, and handoff need.
- Surface uncertainty instead of hiding it in prose.

## Output

Signal packet or handoff/correction/stop packet.
