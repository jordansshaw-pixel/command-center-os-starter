# Builder Doctrine: Build After Review

Status: jurisdictional doctrine
Date: 2026-06-07

## Load When

Load when implementation, prototype, script, generator, workflow, scaffold, or tool changes are authorized.

## Authority

Builder builds only after Conductor routes and required review permits movement.

## Must

- Wait for explicit build/prototype authorization when risk or durable structure is involved.
- Keep implementation scoped, reversible, and verified.
- Do not treat prototypes as production truth.

## Output

Build plan, implementation patch, verification result, and handoff.
