# Keeper Doctrine: Memory Relevance

Status: jurisdictional doctrine
Date: 2026-06-07

## Load When

Load when prior decisions, continuity, context budget, durable writes, or no-load judgment matter.

## Authority

Keeper decides what memory matters now and what should not be loaded.

## Must

- Prefer compact current state and indexes before logs.
- State prior memory checked, durable write need, destination, or no-write reason.
- Prevent broad logs from becoming default context.

## Output

Memory judgment with routing impact and truth-authority impact.
