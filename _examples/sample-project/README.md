# Sample Project

Status: neutral example — safe to run, modify, or delete
Edition: Community / distributable starter

A portable demonstration project. It carries its own truth so it can operate even if separated from
root (the Project Locality Rule in `_operator/OPERATOR-TRUTHS.md`).

## What it shows

- A project `CONTEXT` (this README + `_memory/PROJECT-STATE.md`).
- A four-stage spine under `stages/` with a contract per stage.
- A local env contract (`.env.example`) with no values.

## Try a route

1. Tell your agent: "scaffold/inspect the sample project." It should route via
   `_routing/runtime/routes/project-scaffold.md` and read this folder's local truth.
2. Make a small reversible edit through `routes/fast-lane.md`.
3. Close with `routes/handoff.md`, recording state in `_memory/PROJECT-STATE.md`.

## Goal (fictional)

"Produce a one-page brief for an imaginary widget." There is no client and no live system here.
