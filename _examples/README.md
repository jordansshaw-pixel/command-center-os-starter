# Examples

Neutral, data-free examples that demonstrate how the OS operates. Nothing here is a real project.

- `sample-project/` — a portable project workspace with a four-stage spine
  (`01_intake → 02_judgment → 03_output → 04_handoff`). Use it to prove a route end-to-end
  (project-scaffold → fast-lane → handoff) before adding your own projects.

Add your own project folders at the repo root, not inside `_examples/`.
